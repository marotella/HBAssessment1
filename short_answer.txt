4) Describe what git is and what it is used for.

Your answer: git is a system of version control for a directory/repository on your device. It allows a user to save, change, edit, revert, and branch on a directory, instead of having to create multiple files to track changes.

8) What is the difference between `git add` and `git commit`?

Your answer:
Git add states the changes you are making on the current working version while git commit saves the changes to that version.

------------------
9) What does the `git push` command do?

Your answer:
Git push allows you to upload the local version of your repository to a remote server

------------------
10) When would you use the `git clone` command?

Your answer:
You would use the git clone command when you want to make a copy of a repository that is remote on your device locally, versus working on a branch or fork of the repo.
